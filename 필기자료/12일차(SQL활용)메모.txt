SELECT문

DESCRIBE 테이블명; ->  테이블 정의 상세하게 확인 DESC(동의어)

데이터 조회방법
SELECT [ALL | DISTINCT] FROM 테이블명;
	- ALL : 전체데이터 조회(중복되는 값 모두 조회)
	- DISTINCT : 중복되는 값 제외하고 조회
		- DISTINCT 컬럼1,컬럼2 ~ : 컬럼1+컬럼2 조합된 결과에서 중복제외(두 값이 모두 중복되어야 제외)

관련용어
	- 셀렉션 : 조회된 행
	- 프로젝션 : 조회된 컬럼
	- 조인 : 테이블간 연결

별칭(AS)
컬럼명 [AS] "별칭"
	- AS, "" 생략가능 -> 단, ""은 띄어쓰기가 없을때만 생략가능
	- 테이블 "별칭" -> 반드시 AS생략
	- MySQL -> "",'' 모두 사용 가능 | 테이블 AS "별칭" 사용가능

NVL
	- NVL(컬럼명,대체값); -> NULL값이 있으면 대체값 입력

ORDER BY
	- ORDER BY 기준컬럼명 [ASC(기본값) | DESC]		// 컬럼명을 기준으로 정렬
	- 다수의 정렬조건을 적용하여 정렬
		- ORDER BY 1차정렬컬럼 [ASC(기본값) | DESC], 2차정렬컬럼 [ASC(기본값) | DESC], ...	

WHERE
	- SELECT ... FROM ... WHERE 조건식
	- 조건식 : 비교연산자, 논리연산자, 집합연산자
		- 비교연산자 : <, >, <=, >=, =, ( !=, <>, ^=)
		- 논리연산자 : AND, OR, NOT
		- 집합연산자 : 교집합(INTERSECT,IN), 합집합(UNION), 차집합(MINUS)
