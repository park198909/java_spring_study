CSS
HTML에서 CSS를 적용하는 방법
1. style 태그안에 스타일 속성을 적용'
2. style 속성
3. link 외부 스타일 파일을 연결
	(.css)

선택자
1. 선택자의 종류
1) 태그명으로 선택
	body
	div
	p
	* 모든 태그 - 공통 글꼴 색상, 글꼴 종류... 
	
2) 클래스명으로 선택 : 클래스는 여러개를 동시에 선택할수 있고 class="class1 class2" 중 class2가 우선순위가 높다.
	.클래스명
3) 아이디명으로 선택
	#아이디명

적용 우선순위
가장 많은 선택범위 < 가장 좁은 선택범위

태그 선택자 < 클래스 선택자 < 아이디 선택자 < 스타일 속성

*** !important -> 강제 우선순위 적용 : 사용하지 않는 걸 추천(꼬일 가능성 매우 높음)

4) 조상 자손 선택자

조상...자손 { ... }

5) 부모 자식 선택자
부모>자식 { ... }

6) 형제 선택자
선택자~선택자 : 오른쪽 모든 형제
선택자+선택자 : 오른쪽 바로 옆 형제

7) 그룹 선택자
선택자,선택자,선택자 { ... }

8) 가장 클래스 선택자
상태를 선택
	선택자:hover{ 마우스가 올라간 상태 }
	선택자:focus{ 커서가 올라간 상태 }
	선택자:selected{ 선택된 상태 }
	선택자:checked{ checkbox,radio가 체크된 상태 }
		선택자1:checked+선택자2{ 선택자1이 체크되었을 때 인접형제인 선택자2에 적용 }
	선택자:read-only{ 읽기 전용 상태 }

	선택자:not(선택자2) { 선택자 중 선택자2를 제외하고 모두선택 }

	선택자:first-child { 선택자 중 첫번째 자식 선택 }
	선택자:last-child { 선택자 중 마지막 자식 선택 }
	선택자:nth-child(숫자) { 1번 부터 시작하는 숫자번째 요소를 선택 }
	선택자:nth-child(수식) { 수식에 해당하는 요소를 선택 }
		1,3,5,7,9... => 2n+1 : 이 경우 n=0부터 시작
		2,4,6,8,10... => 2n : 이 경우 n=1부터 시작
	선택자:first-of-type {}		//child의 종속관계를 제거하기위해 사용
	선택자:last-of-type {}		//child의 종속관계를 제거하기위해 사용
	선택자:nth-of-type(숫자,수식) {}	//child의 종속관계를 제거하기위해 사용

	선택자:after { content='선택자의 하위요소 가장 끝에 내용추가'  }		//content내용필수
	선택자:before { content='선택자의 하위요소 가장 처음에 내용추가' }		//content내용필수

9)속성명으로 선택
선택자[속성명="값"] { 적용할 내용 }
선택자[속성명*="값"] { 값이 포함된 모든속성에 적용할 내용 }
선택자[속성명^="값"] { 값으로 시작되는 속성에 적용할 내용 }
선택자[속성명$="값"] { 값으로 끝나는 속성에 적용할 내용 }
